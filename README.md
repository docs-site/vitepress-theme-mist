## git提交记录规范

> git 提交信息 emoji： [https://gitmoji.dev/](https://gitmoji.dev/)

遵循以下格式：

```bash
<type>(<scope>): <subject>

# type，表示类型，可选fix、feat等
# scope，影响的范围，看大多数用的都是()
# subject，提交信息
```
